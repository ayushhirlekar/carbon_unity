# CarbonUnity

Web3-based carbon credit marketplace connecting farmers and buyers.

## Project Structure
- `/landing-page` - Static HTML landing page
- `/frontend` - React application
- `/backend` - Node.js API server
- `/database` - Supabase schema and migrations
- `/smart-contracts` - Solidity contracts (Sepolia)
- `/docs` - Documentation

## Tech Stack
- Frontend: React + MetaMask
- Backend: Node.js + Express
- Database: Supabase (PostgreSQL)
- Blockchain: Ethereum Sepolia + Solidity
- Authentication: Wallet-based (MetaMask signatures)
